## Projetos principais

### Teseo `[em andamento]`
SaaS fullstack para TCCs e dissertações. Frontend em Next.js, backend em FastAPI, pipeline RAG com pgvector para busca semântica, memória persistente por projeto, automação ABNT e prevenção de plágio.

`Next.js` `React` `FastAPI` `PostgreSQL` `pgvector` `RAG` `LLMs`

---

### TrackPet — TCC
Geolocalização em tempo real entre dono e quem encontrou o pet, com mensageria assíncrona para notificações. Backend com Arquitetura Hexagonal + DDD, APIs RESTful e modelagem relacional rigorosa.

`Java` `Angular` `Spring Boot` `RabbitMQ` `REST API` `DDD` `MySQL`

---

### FPO Automation — P&D UniFio
Migração de formulação MATLAB proprietária (498 linhas) para Python open-source (296 linhas). Automação completa de geração de modelos de Fluxo de Potência Ótimo validada no IEEE 14-bus system.

**−40% LOC · 100% automação de geração de modelos**

`Python` `Pyomo` `SymPy` `SciPy` `Pandas` `MySQL`

---

### Modelo de Inadimplência & Clusterização
Pipeline ML end-to-end para risco de crédito com LightGBM + validação cruzada e segmentação K-Means integrada a agentes LLM para interpretação dos clusters.

`Python` `LightGBM` `Scikit-learn` `LangChain` `K-Means`

---

### Agentes LLM com RAG & MCP
Agentes com RAG para consulta sobre bases proprietárias e integração MCP com ferramentas externas. Documentação técnica de skills para uso em produção.

`Python` `LangChain` `LlamaIndex` `MCP`

---

### Sistema de Gestão — Fito Sollos
Sistema end-to-end com modelagem de banco conforme regras de negócio, backend Java integrado a frontend React.

`Java` `React` `MySQL`
